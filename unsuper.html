Unsupervised Learning


Unsupervised learning is a type of Machine learning where the model works with unlabeled data, aiming to identify patterns or structures within the data. It groups similar data points (clustering) or reduces data complexity (dimensionality reduction) without predefined outcomes. Examples include customer segmentation and anomaly detection.
